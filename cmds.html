Commands

  • ~j purge (amount) - Deletes a specified number of messages from the channel.
    Usage: ~j purge 10 (Deletes the last 10 messages.)

  • ~j av (user) - Displays the avatar of the mentioned user.
    Usage: ~j av @username (Shows the avatar of the tagged user.)

  • ~j lock (blankOR#) - Locks a specified channel to restrict access.
    Usage: ~j lock #general or ~j lock (Locks the "general" channel or the current channel.)

  • ~j unlock (blankOR#) - Unlocks a specified channel to allow access.
    Usage: ~j unlock #general or ~j unlock (Unlocks the "general" channel or the current channel.)

  • ~j cat - Displays a random cat image.
    Usage: ~j cat (Shows a cute cat picture.)

  • ~j meme - Posts a random meme.
    Usage: ~j meme (Shares a funny meme in the chat.)

  • ~j blip (blankOR@) - Sends a notification to a specified user or provides a generic message.
    Usage: ~j blip @username or ~j blip (Notifies the mentioned user or sends a general message.)

  • ~j currency - Displays information about the server's currency system.
    Usage: ~j currency (Shows details on how to earn and use currency.)

  • ~j extracash - Grants extra currency to the user.
    Usage: ~j extracash (Adds additional currency to the user's balance.)

  • ~j bal (blankOR@) - Checks the balance of the user or a mentioned user.
    Usage: ~j bal @username or ~j bal (Displays the balance of the mentioned user or the user who issued the command.)

  • ~j help - Provides a list of commands and their descriptions.
    Usage: ~j help (Displays a help message with available commands.)

  • ~j commands - Lists all available commands for Jaren.
    Usage: ~j commands (Shows a complete list of commands.)

  • ~j support - Provides information on how to get support for using Jaren.
    Usage: ~j support (Displays links or information for support resources.)

  • ~j affiliate - Displays affiliate information or links.
    Usage: ~j affiliate (Shows details about Jaren’s affiliate program.)

  • ~j up - Checks for updates or upgrades available for Jaren.
    Usage: ~j up (Displays current version information and available updates.)

  • ~j prefix (boolean) - Changes or checks the command prefix for Jaren.
    Usage: ~j prefix true or ~j prefix (Sets a new command prefix or displays the current prefix.)
